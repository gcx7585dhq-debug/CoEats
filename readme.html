🍱 CoEats 公司訂餐助手

基於 LIFF 與 Vue 3 的智慧化行政訂餐解決方案。

01 核心功能

✨ AI 智慧識別

利用 Tesseract.js 技術，行政人員只需拍攝菜單照片，系統即可自動提取店名、菜名與價格。

📲 LINE 一鍵點餐

深度整合 LINE LIFF SDK，點擊按鈕即可在對話視窗自動送出點餐訊息，無需手動打字。

🛠️ 店家管理系統

支援即時編輯店名、新增/刪除菜色，所有資料儲存於 LocalStorage，具備基本的持久化能力。

♿ 無障礙優化

遵循 Web 標準開發，修復所有 Form Label 與 Button Text 警告，確保流暢的開發與使用體驗。

02 快速開始

  1. 將專案上傳至 GitHub 並啟用 GitHub Pages。
  2. 前往 LINE Developers 建立 LIFF App。
  3. 將 Endpoint URL 指向你的 GitHub Pages 網址。
  4. 修改 index.html 末尾的 YOUR_LIFF_ID。

03 技術棧

Vue 3 (Composition API) Tailwind CSS Tesseract.js LINE LIFF SDK
